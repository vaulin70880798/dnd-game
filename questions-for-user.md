# שאלות להשלמת הטמעה מלאה

1. האם תרצה שאסיר בסוף את `textOriginal` (רוסית) ואשאיר רק עברית בקובץ הפסקאות?
2. האם יש לך את הטקסט המקורי המדויק של פסקאות `175`, `264`, `328` כדי להחליף את ה־placeholders?
3. האם אתה רוצה לבחור במסך יצירת הדמות בין מצב לוחם/קוסם (עם נוסחאות שונות למיומנות) כבר עכשיו?
4. האם יש לך גם את "ספר הלחשים" כדי למבנה spells לקובץ נתונים נפרד ולא רק בטקסט הפסקאות?
5. האם להמשיך עכשיו למעבר ידני מלא על כל 27 פסקאות הקרב שכבר הובנו אוטומטית ולדייק כל כלל מיוחד?
