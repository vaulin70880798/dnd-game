# איך להוסיף עוד פסקאות לספר

1. הוסף פסקה חדשה ל־`data/paragraphs.json` עם מזהה מספרי ייחודי (`id`).
2. ודא שלכל `choice.target` יש פסקה קיימת.
3. אם הפסקה כוללת קרב, ודא שכל אויב קיים ב־`data/enemies.json`.
4. אם הפסקה כוללת חפצים (`effects`), ודא שכל `itemId` קיים ב־`data/items.json`.
5. חבר את הפסקה ל־`locationId` קיים, או הוסף צומת חדשה ב־`data/mapNodes.json`.
6. הרץ `npm run validate:data` ובדוק שאין שגיאות.
7. הרץ `npm run build` לווידוא סופי לפני פריסה.

טיפ: עדיף להכניס את הספר באצוות של 20–40 פסקאות, להריץ ולידציה, ורק אז להמשיך לאצווה הבאה.
